В папке расположена программа "Regim_mlab", реализующая расчет режима
сети 6-10 кВ произвольной конфигурации (в том числе замкнутой
конфигурации с множественными источниками питания, но одного номинального 
напряжения) по методу контурных уравнений. Реализована работа с xls файлами для
ввода и вывода результатов расчета.

Для запуска программы необходимо:
1. Скопировать папку Regim_Mlab на жесткий диск
2. Открыть файл main в Matlab;
3. Запустить файл main в Matlab;
4. Выбрать рабочий каталог в Matlab;
5. Исходные данные и результаты хранятся в формате xls в папке Excel Data
 под именем "Исходные_данные.xls", "Результаты.xls";
*В архиве хранятся дополнительные тестовые схемы для их использования
в программе необходимо переименовать соответствующий файл.

В папке расположены 2 m-файла:
1) main.m - Matlab скрипт - является управляющим блоком для расчета режима, 
использует окружение для создание модели сети, графа, представляющего сеть, 
графического вывода схемы и расчета режима сети.
2) go.m - Matlab скрипт – обеспечивает запись каталога и подкаталогов с файлом 
main в пути поиска Matlab.
Также в папке расположены 5 подкаталогов:
1) Excel Data - папка содержит файлы исходных данных и результатов excel
2) FGraphs - папка содержит m-файлы с функциями и классом для работы с графом
3) FRegim - папка содержит файлы с функциями для организации расчета режима
4) Model_Data - папка содержит m-файлы с классами и функциями для создания
модели, ввода и вывода исходных данных
5) Other - папка содержит m-файлы, реализующие особые возможности, не связанные
с основными задачами системы (расчет и оптимизация режима сети 6-10 кВ).
На момент создания первой версии (05_01_18) в папке расположен единственный
файл.
