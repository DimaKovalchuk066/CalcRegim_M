Папка EnumType содержит m-файлы с enumeration классами.
На момент создания первой версии (05_01_18) в папке расположено 6
файлов:
1) bTip - класс определяет перечисления типов ветвей.
2) cmStat - класс определяет перечисления типов положений КА.
3) cmTip - класс определяет перечисления типов КА.
4) methTip - класс определяет метод ввода исходных данных. 
5) nTip - класс определяет перечисления типов узлов.
6) Tip - Класс определяет перечисления типов элементов 
модели для класса CData.
