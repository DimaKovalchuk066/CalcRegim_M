Папка Model_Data содержит m-файлы с классами и функциями для создания
модели, ввода и вывода исходных данных.
На момент создания первой версии (05_01_18) в папке расположено 3
файла:
1) CData - Value-класс организует работу с исходными данными 
(ввод данных - на момент создания реализован ввод данных из 
excel и первичная обработка). Предполагается реализация 
в данном классе первичного дорасчета исходных данных для
к необходимому в классе-модели виду.
2) CModelRS  Handle-класс читает из указанного объекта с данными 
по схеме распределительной электричекой сети, обеспечивает их 
хранение и предоставление доступа. Осуществляет дорасчет 
дополнительных параметров, таких как токи ветвей, 
потери мощности и напряжений.
3) fExcelOut - Matlab функция, обспечивающая вывод 
исходных данных из модели, представленной CModelRS в xls файл.

В каталоге расположен 1 подкаталог в котором собраны все,
используемые в программе enumeration-классы.
