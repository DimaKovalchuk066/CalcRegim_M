папка содержит файлы исходных данных и результатов excel.

В папке расположено множестов xls файлов из которых рабочими являются 2:
1) "Исходные_данные.xls" содержит на 2 листах исходные данные об узлах и ветвях
схемы.
2) "Результаты.xls" содержит на 3 листах результаты расчета режима по узлам и
ветвям схемы.
